私は誰？

私はスリランカ出身のウェブ開発者で、 現在日本で留学しています。 見た目が美しく、使いやすく、そして機能的なウェブサイトを作ることに情熱を持っています。 ビジネスや個人がオンラインでアイデアを実現できるようサポートし、 各プロジェクトがその独自の目標を反映することを目指しています。

何ができるのか？

Java

JAVAはクラスベースのオブジェクト指向、マルチスレッド、ガベージコレクション、コンポーネントベース、分散コンピューティングといった特徴を持ち機能的な プログラムを作成することができます。

HTML, CSS, JS

最新のウェブ技術を活用し、 あなたの夢のアイデアを現実にし ます。

UI and UX

個人のブログから複雑なビジネス プラットフォームまで、真に共感 できるウェブ体験を構築します。

なぜ私と働くのか？

信頼性

ハンドメイドプロジェクト

すべてのプロジェクトはあなた の意見とアイデアに合わせてカ スタマイズされます。
